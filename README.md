The original data files are "raw data of the control group.xlsx" and "Raw data of the experimental group.xlsx".
The versions after processing the original data are "data.omv" and "pyinput.xlsx", which are used for data analysis in Python and Jamovi, respectively.
"Interview Outline" contains the interview guide.
"Mediation analysis code" contains Python mediation analysis code.
"I1-I10.docx" are interview data files.
"Data Collection Procedures and Variable Overview and Variable Coding Scheme.docx" contains variable definitions, scale sources, data collection procedures, and qualitative coding schemes.
